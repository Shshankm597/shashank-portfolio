My learnings of first two nodeJS modules: Readline and CHALK

by Shashank Mishra, updated on November,2020
5 min read

This is a dummy blog.

The readline module provides an interface for reading data from a Readable stream (such as process. stdin ) one line at a time. It can be accessed using: const readline = require('readline');

If you use Node. js, you're probably familiar with the Chalk module. It's one of my favorite npm modules; and, is generally one of the first modules that I install when I'm playing around with Node. js. The Chalk module allows you to add styles to your terminal output.
